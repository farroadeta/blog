### 一、 基础单音教学法术语
1.  **演示法 / 示范法**
    *   **术语**：**直观演示法**。这是最基础的教学方法。
    *   **相关术语**：**口形示范**、**舌位图演示**、**板书演示**（如用五度标记法画声调）。

2.  **语音学辅助法**
    *   **术语**：**语音学讲解法** 或 **理性主义方法**。这种方法侧重于语言的知识性和理解。
    *   **相关术语**：**发音部位**、**发音方法**、**清浊音**、**送气/不送气**。

3.  **带音法**
    *   **术语**：**带音法** 或 **引导法**。这是一个非常经典的语音教学专用术语。

4.  **模仿法**
    *   **术语**：**模仿法** 或 **跟读法**。这是最古老也是最核心的方法之一，属于**听说法**的代表性技巧。
    *   **相关术语**：**集体模仿**、**个别模仿**。

### 二、 声调教学法术语

1.  **五度标记法 & 手势法**
    *   **术语**：
        *   **五度标记法**：这是语言学中描写声调调值的标准术语。
        *   **手势法**：或具体称为**声调手势指挥法**。这是教学中将声调"体势化"的重要手段。

2.  **唱调法**
    *   **术语**：**唱调法** 或 **声调音乐化训练**。

3.  **对比练习法**
    *   **术语**：**最小对立对对比法** 或 **辨音辨调练习**。
    *   **相关术语**：**最小对立对**（如"妈mā"和"麻má"）、**双音节声调组合练习**（如"妈妈"、"麻花"、"马路"）。

### 三、 语流教学法术语

1.  **变调训练**
    *   **术语**：**语流音变教学** 或 **连读变调规则教学**。
    *   **相关术语**：**第三声变调**、**"一"和"不"的变调**、**轻声**。

2.  **轻声 & 儿化训练**
    *   **术语**：**轻声教学**、**儿化韵教学**。

3.  **节奏与韵律训练**
    *   **术语**：**韵律教学** 或 **超音段音位教学**（因为声调、节奏、重音等都是超出单个音素的成分）。
    *   **相关术语**：**停延**、**重音**、**节律**。

### 四、 综合与趣味教学法术语

1.  **绕口令**
    *   **术语**：**绕口令** 或 **急口令**。在教学中，它属于**趣味练习**或**强化练习**的一种。

2.  **朗读与朗诵**
    *   **术语**：**朗读法**、**朗诵法**。

3.  **唱歌学汉语**
    *   **术语**：**歌曲教学法** 或 **音乐介入法**。

4.  **游戏法**
    *   **术语**：**游戏教学法** 或 **互动式语音练习**。
    *   **相关术语**：**听辨游戏**、**发音游戏**。

5.  **技术辅助法**
    *   **术语**：**技术辅助语音教学** 或 **多媒体语音教学**。
    *   **相关术语**：**语音分析软件**、**可视化语音训练**。

### 五、 宏观教学原则与流派术语

这些方法背后体现了一些更大的教学理念和流派：

*   **对比分析**：在预测教学难点时，系统地比较汉语与学生母语的语音系统。
*   **偏误分析**：系统地分析学生产生的语音错误，并据此进行针对性教学。
*   **循序渐近**：教学原则，从**音素**（单个音）到**音节**（字），再到**词语**和**句子**。
*   **交际法**：强调语音教学要为交际服务，避免纯粹的机械操练，而是在有意义的对话中练习。
*   **全身反应法**：其中的**手势法**就与此流派理念相通，通过身体动作辅助记忆。
*   **听说法**：**模仿法**、**重复操练**是其核心手段，强调通过刺激-反应形成习惯。

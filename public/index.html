#语言学

3.1 四要素的别义功能

一.语音四要素（P69）

  • 音高：由频率决定。大、长、粗、厚、松的发音体振动慢，所以频率低，反之，频率就高。
  • 音强：由振幅决定。振幅的大小决定于使发音体振动的外力的大小。
  • 音长：由时间决定。发音体振动的时间决定音长。
  • 音质：是声音的个性特质。发音体，发音方法，共鸣器的差异都会形成不同的音质。音质是四要素中最重要的一点。

四要素的别义

  1. 普通话四声调的差别主要由音高 决定的。
  2. 语言中语调的差别主要由 音高决定的。
  3. 汉语普通话中的轻声就是音强在起作用。
  4. 英语中的轻重音的差别是音强 在起作用。
  5. 英语中长音和短音的差别是音长在起作用。
  6. “请勿高声说话”的“高”指声音的响度，主要是音强在起作用。
  7. 声音频率是倍数关系的波形成的音叫乐音。 反之，频率不是倍数关系的波形成的音叫噪音。 两者兼之的是混合音。 语音中的元音属于乐音；清辅音 属于噪音；浊辅音属于混合音。

3.2 语音区位特征

一、元音和辅音的区别（P71）

  1. 呼出气流是否受阻，元音不受阻，辅音受阻；
  2. 发音器官的紧张程度是否均衡，元音均衡，辅音不均衡；（受阻部分紧张，其余部分不紧张）
  3. 呼出气流的强弱不同，元音较弱，辅音较强。
  4. 声带振动不同，发元音，浊辅音时（m、n、l、 r、ŋ）声带振动（乐音），清辅音声带不振动（噪音）

二、辅音的语音特征（P74）

主要从发音部位和发音方法两个角度分析，重点掌握普通话21个声母的语音特征。

  • 发音部位：双唇、唇齿、舌尖前、舌尖中、舌尖 后、舌面、舌根；
  • 发音方法1：气流受阻除阻情况，有塞音、塞擦音、 擦音、鼻音、边音；
  • 发音方法2：声带振动情况，有清音、浊音；
  • 发音方法3：气流强弱情况，有送气音、不送气；

三、音标和国际音标

广义音标指一切标记语音的符号。汉语注音字 母、汉语拼音、日语假名。 狭义音标指标记音素的音标，通常就是指国际 音标。 国际音标是国际语音协会于1888年制定的，其 后基本上是10年修订补充一次，主要特点：

  1. 音标和音素有一一对应的关系；
  2. 国际音标符号采用世界通用的拉丁字母。
  3. 国际音标的数量完备，可以满足世界主要 语言（方言）的记音。

用国际音标标注汉语音节，分为宽式音标和严 式音标。宽式音标记音位，严式音标记音质。

#语言学

1.1 什么是语言

一、动物是否有语言

1. 动物传递信息的特点

  • 本能、条件反射、现场刺激
  • 信息有限
  • 无法拆卸和组装

2. 语言的定义

  • 广义概念：包括人类语言和动物信息传递系统

  • 狭义概念：专指人类语言

    （动物没有狭义语言；体态语和文字不属于狭义语言）

二、人类的语言

1. 人类语言的特点

  1. 功能开放
  2. 结构灵巧
  3. 后天习得

2. 基础知识点

  • (1) 幼儿习得语言的三种假说

    模仿说、天赋说、认知说

  • (2) 儿童语言习得的五个阶段

    咿呀阶段 → 独词句阶段 → 双词句阶段 → 电报体阶段 → 接近成人阶段

  • (3) 大脑分工

    左半球：语言与抽象思维  
右半球：艺术与形象思维

1.2 对语言的几个重要认识

1. 符号（Sign）

  • 定义：事物的标记，包含形式（能指）和意义（所指）
  • 特征：形式与意义的联系由社会约定（无必然关联）

2. 符号类型

  • 按感知方式：视觉符号、听觉符号、触觉符号

  • 按符号性质：

    - 图像（Icon）：相似性符号（如照片）  
- 直示（Index）：因果/邻近关系符号（如烟→火）  
- 标记（Symbol）：任意约定符号（如语言）

3. 语言符号的特点

  • 基本特征：语音与语义的统一体

  • 主要特点：

    - 任意性（形式与意义无必然联系）  
- 线条性（符号线性排列）

  • 辩证关系：

    - 任意性 vs 强制性（社会约定后不可随意改变）  
- 任意性 vs 理据性（部分词汇有构词理据）

1.3 语言学的发展和分类

一、兴起与发展

  5. 当代：形式语言学、功能语言学、认知语言学

利奇的意义七分法

内容

  1. conceptual meaning理性意义（概念意义）是一个词的核心意义，也是基本意义。女人：就是成年的女性。
  2. Associated meaning内涵意义（联想意义）女人：好动情、有同情心、敏感、反复无常等联想，联想义不稳定，有开放性和模糊性的特点，龙在中国至高无上，在圣经中，是邪恶的动物。
  3. social meaning社会意义（风格意义）在不同的交际场合带来的风格或语体的附加意义。汉语“情感、情愫”语体色彩不同，英语中的马:horse一般用语,steed诗歌用语,nag俚语。
  4. emotive meaning情感意义情感色彩，逝世/死；闭上你的臭嘴/请不要讲话。可以表达不同的情感态度。
  5. reflective meaning反映意义（反射意义）由一个词语联系起另外一种意义。8888的数字受欢迎，带4的号码不受欢迎，就是含有反映意义。
  6. collocative meaning搭配意义prettygirl，hanndsomeboy；汉语的“吠、鸣、嚎、啸”都有“叫”的意思，但搭配有差异。
  7. thematic meaning主题意义（主位意义）通过信息的顺序强调所表达的意义。如“武松打死了老虎”和“老虎被武松打死了”，前句是讲武松干什么？后句是讲老虎怎么样？有不同的主题意义。

评析

  1. 分类没有标准，有举例性质，类型有点乱。有的是词义，有的是句义：有的是语言义，有的是言语义：有的是理性义，有的是非理性义：
    • 理性义、内涵义、社会义、情感义属于词义
    • 搭配义、主题义属于句义
    • 概念义属于语言义，属于理性意义；内涵义、社会义、情感义属于言语义，属于非理性意义。
  2. 反映意义、主题意义等内涵不够明确。

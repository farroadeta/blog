#语言学

3.1 四要素的别义功能

一.语音四要素（P69）

  • 音高：由频率决定。大、长、粗、厚、松的发音体振动慢，所以频率低，反之，频率就高。
  • 音强：由振幅决定。振幅的大小决定于使发音体振动的外力的大小。
  • 音长：由时间决定。发音体振动的时间决定音长。
  • 音质：是声音的个性特质。发音体，发音方法，共鸣器的差异都会形成不同的音质。音质是四要素中最重要的一点。

四要素的别义

  1. 普通话四声调的差别主要由音高 决定的。
  2. 语言中语调的差别主要由 音高决定的。
  3. 汉语普通话中的轻声就是音强在起作用。
  4. 英语中的轻重音的差别是音强 在起作用。
  5. 英语中长音和短音的差别是音长在起作用。
  6. “请勿高声说话”的“高”指声音的响度，主要是音强在起作用。
  7. 声音频率是倍数关系的波形成的音叫乐音。 反之，频率不是倍数关系的波形成的音叫噪音。 两者兼之的是混合音。 语音中的元音属于乐音；清辅音 属于噪音；浊辅音属于混合音。

3.2 语音区位特征

一、元音和辅音的区别（P71）

  1. 呼出气流是否受阻，元音不受阻，辅音受阻；
  2. 发音器官的紧张程度是否均衡，元音均衡，辅音不均衡；（受阻部分紧张，其余部分不紧张）
  3. 呼出气流的强弱不同，元音较弱，辅音较强。
  4. 声带振动不同，发元音，浊辅音时（m、n、l、 r、ŋ）声带振动（乐音），清辅音声带不振动（噪音）

二、辅音的语音特征（P74）

主要从发音部位和发音方法两个角度分析，重点掌握普通话21个声母的语音特征。

  • 发音部位：双唇、唇齿、舌尖前、舌尖中、舌尖 后、舌面、舌根；
  • 发音方法1：气流受阻除阻情况，有塞音、塞擦音、 擦音、鼻音、边音；
  • 发音方法2：声带振动情况，有清音、浊音；
  • 发音方法3：气流强弱情况，有送气音、不送气；

三、音标和国际音标

广义音标指一切标记语音的符号。汉语注音字 母、汉语拼音、日语假名。 狭义音标指标记音素的音标，通常就是指国际 音标。 国际音标是国际语音协会于1888年制定的，其 后基本上是10年修订补充一次，主要特点：

  1. 音标和音素有一一对应的关系；
  2. 国际音标符号采用世界通用的拉丁字母。
  3. 国际音标的数量完备，可以满足世界主要 语言（方言）的记音。

用国际音标标注汉语音节，分为宽式音标和严 式音标。宽式音标记音位，严式音标记音质。

#语言学

1.1 什么是语言

一、动物是否有语言

1. 动物传递信息的特点

  • 本能、条件反射、现场刺激
  • 信息有限
  • 无法拆卸和组装

2. 语言的定义

  • 广义概念：包括人类语言和动物信息传递系统

  • 狭义概念：专指人类语言

    （动物没有狭义语言；体态语和文字不属于狭义语言）

二、人类的语言

1. 人类语言的特点

  1. 功能开放
  2. 结构灵巧
  3. 后天习得

2. 基础知识点

  • (1) 幼儿习得语言的三种假说

    模仿说、天赋说、认知说

  • (2) 儿童语言习得的五个阶段

    咿呀阶段 → 独词句阶段 → 双词句阶段 → 电报体阶段 → 接近成人阶段

  • (3) 大脑分工

    左半球：语言与抽象思维  
右半球：艺术与形象思维

1.2 对语言的几个重要认识

1. 符号（Sign）

  • 定义：事物的标记，包含形式（能指）和意义（所指）
  • 特征：形式与意义的联系由社会约定（无必然关联）

2. 符号类型

  • 按感知方式：视觉符号、听觉符号、触觉符号

  • 按符号性质：

    - 图像（Icon）：相似性符号（如照片）  
- 直示（Index）：因果/邻近关系符号（如烟→火）  
- 标记（Symbol）：任意约定符号（如语言）

3. 语言符号的特点

  • 基本特征：语音与语义的统一体

  • 主要特点：

    - 任意性（形式与意义无必然联系）  
- 线条性（符号线性排列）

  • 辩证关系：

    - 任意性 vs 强制性（社会约定后不可随意改变）  
- 任意性 vs 理据性（部分词汇有构词理据）

  4. 语言作为符号系统

利奇的意义七分法

内容

  1. conceptual meaning理性意义（概念意义）是一个词的核心意义，也是基本意义。女人：就是成年的女性。
  2. Associated meaning内涵意义（联想意义）女人：好动情、有同情心、敏感、反复无常等联想，联想义不稳定，有开放性和模糊性的特点，龙在中国至高无上，在圣经中，是邪恶的动物。
  3. social meaning社会意义（风格意义）在不同的交际场合带来的风格或语体的附加意义。汉语“情感、情愫”语体色彩不同，英语中的马:horse一般用语,steed诗歌用语,nag俚语。
  4. emotive meaning情感意义情感色彩，逝世/死；闭上你的臭嘴/请不要讲话。可以表达不同的情感态度。
  5. reflective meaning反映意义（反射意义）由一个词语联系起另外一种意义。8888的数字受欢迎，带4的号码不受欢迎，就是含有反映意义。
  6. collocative meaning搭配意义prettygirl，hanndsomeboy；汉语的“吠、鸣、嚎、啸”都有“叫”的意思，但搭配有差异。
  7. thematic meaning主题意义（主位意义）通过信息的顺序强调所表达的意义。如“武松打死了老虎”和“老虎被武松打死了”，前句是讲武松干什么？后句是讲老虎怎么样？有不同的主题意义。

评析

  1. 分类没有标准，有举例性质，类型有点乱。有的是词义，有的是句义：有的是语言义，有的是言语义：有的是理性义，有的是非理性义：
    • 理性义、内涵义、社会义、情感义属于词义
    • 搭配义、主题义属于句义
    • 概念义属于语言义，属于理性意义；内涵义、社会义、情感义属于言语义，属于非理性意义。
  2. 反映意义、主题意义等内涵不够明确。

  2. 语音学辅助法

    • 术语：语音学讲解法 或 理性主义方法。这种方法侧重于语言的知识性和理解。
    • 相关术语：发音部位、发音方法、清浊音、送气/不送气。

  3. 带音法

    • 术语：带音法 或 引导法。这是一个非常经典的语音教学专用术语。

  4. 模仿法

    • 术语：模仿法 或 跟读法。这是最古老也是最核心的方法之一，属于听说法的代表性技巧。
    • 相关术语：集体模仿、个别模仿。

二、 声调教学法术语

  1. 五度标记法 & 手势法

    • 术语：
      • 五度标记法：这是语言学中描写声调调值的标准术语。
      • 手势法：或具体称为声调手势指挥法。这是教学中将声调“体势化”的重要手段。

  2. 唱调法

    • 术语：唱调法 或 声调音乐化训练。

  3. 对比练习法

    • 术语：最小对立对对比法 或 辨音辨调练习。
    • 相关术语：最小对立对（如“妈mā”和“麻má”）、双音节声调组合练习（如“妈妈”、“麻花”、“马路”）。

三、 语流教学法术语

  1. 变调训练

    • 术语：语流音变教学 或 连读变调规则教学。
    • 相关术语：第三声变调、“一”和“不”的变调、轻声。

  2. 轻声 & 儿化训练

    • 术语：轻声教学、儿化韵教学。

  3. 节奏与韵律训练

    • 术语：韵律教学 或 超音段音位教学（因为声调、节奏、重音等都是超出单个音素的成分）。
    • 相关术语：停延、重音、节律。

四、 综合与趣味教学法术语

  1. 绕口令

    • 术语：绕口令 或 急口令。在教学中，它属于趣味练习或强化练习的一种。

  2. 朗读与朗诵

    • 术语：朗读法、朗诵法。

  3. 唱歌学汉语

    • 术语：歌曲教学法 或 音乐介入法。

  4. 游戏法

    • 术语：游戏教学法 或 互动式语音练习。
    • 相关术语：听辨游戏、发音游戏。

  5. 技术辅助法

    • 术语：技术辅助语音教学 或 多媒体语音教学。
    • 相关术语：语音分析软件、可视化语音训练。

五、 宏观教学原则与流派术语

这些方法背后体现了一些更大的教学理念和流派：

  • 对比分析：在预测教学难点时，系统地比较汉语与学生母语的语音系统。
  • 偏误分析：系统地分析学生产生的语音错误，并据此进行针对性教学。
  • 循序渐近：教学原则，从音素（单个音）到音节（字），再到词语和句子。
  • 交际法：强调语音教学要为交际服务，避免纯粹的机械操练，而是在有意义的对话中练习。
  • 全身反应法：其中的手势法就与此流派理念相通，通过身体动作辅助记忆。
  • 听说法：模仿法、重复操练是其核心手段，强调通过刺激-反应形成习惯。

#第二语言习得

一般因素

年龄

  • 一般认为，儿童学第二语言比成人快，年龄愈小愈有优势。
  • 关键期假说
    • 关键期，又称：临界期，是2岁开始至青春期（约十二三岁）。
    • 人生发展的某个特定阶段，在这个阶段，人可以在没有外部干预、不需要教授的条件下，轻松快速地习得一门语言。
  • 原因：
    1. 生理原因：关键期之前，大脑更有弹性
    2. 社会原因：
      • 儿童不容易收到同伴压力，自我认识方面没有形成模式化思想，更容易接受外部信息。
      • 儿童情感屏障弱于成年人，不会像成年人那样过于计较他人对自己的态度。
    3. 认知原因：
      • 儿童使用语言习得机制，第二语言习得如同母语习得；成人使用一般的问题解决能力。
      • 儿童在认知水平有限，抽象思维能力尚未形成，学习第二语言时一般不能利用所学的语言规则监控语言输出。

性格

  • 外向型
  • 内向型
  • 焦虑
  • 歧义容忍度、移情、冒险精神、自尊等

认知方式

场独立型和场依存型

  • 场独立型
    • 认识事物很少受环境和他人影响
    • 以分析方式看待事物
  • 场依存型
    • 受环境和他人影响较大
    • 以整体方式看待事物

审慎型和冲动型

又叫：概念化速度，是指在不确定条件下个体做出决定时速度上的差异。

  • 审慎型：进行反映前深思熟虑，仔细考虑所有可能性
  • 冲动型：简短地考察各种可能性后做出迅速做出决定

  2. 语法敏感性
  3. 归纳能力
  4. 机械记忆能力

  2. 对学习目的语的态度
  3. 对语言和学习语言的一般态度

动机

内在动机和外在动机

  • 内在动机
    • 为了从参与活动本身获得愉快和满足
    • 取决于外语学习者的内在需要
  • 外在动机
    • 达到某一目的途径，如外在奖励（父母的赞同、考试取得高分等）、避免惩罚等。
  • 内在动机与长期成功有联系，外在动机与短期成功相关。

融合性动机和工具型动机

  • 融合型动机：对目的语群体充满兴趣，理解目的语群体的文化，希望与目的语群体有更多的接触，甚至希望成为目的语群体的一员
  • 工具型动机：对目的语群体没有兴趣。学习目的语只是为了掌握一个工具，用来提高自己的知识水平，改善社会地位。 可见 [[5.社会文化模式]]

  • 学习时间长＞学习时间短
  • 高级水平＞初级水平
  • 多语和双语学习者＞单语学习者

#第二语言习得

性质

  • 普遍语法由一些普遍的原则、条件和规则组成。
  • 反映了人类语言的本质特征，这些特征构成了学习者的语言习得的起点，即学习者的“初始状态”。学习者的能力就是在初始状态上发展起来的。

构成

  • 所有人类都传承了一套普遍的原则和参数。这些原则和参数构成了人类语言可能具有的形态。
  • 人类语言的核心是“词库”：
    • 词汇语类：包含实词的类别。
    • 功能语类：包含指示词、助词以及抽象的语法特征

原则和参数

原则

高度限定性的：事先规定了有限的规则或限制，限制了哪些规则是可能发生的，哪些规则是不可能发生的。

参数

  • 学习者的“语言能力”是由所在语言中恒定不变的“计算程序”和词库两部分构成。
  • 儿童习得母语无需对复杂而又抽象的语言结构进行分析，只需根据有限的语言输入设定所学语言的参数。
